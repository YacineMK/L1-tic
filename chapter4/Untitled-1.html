le web et internet

- internet est un support de communication qui permet la transmission de données à travers le monde par contre le web est une application spécifique d'internet.

Aspects techniques du Web

Le web est basé sur trois mécanismes qui sont :

  • Url

    qui est l'adresse d'une ressource donnée, unique sur le Web.

  • http

    qui est l'ensemble de règles régissant le transfert de fichiers (texte, images, son, vidéo, et autres fichiers multimédias) sur le Web.

  • Html

    est un langage de description de pages Web qui permet de présenter les documents hypertextes destinés à être affichés sur le navigateur.

Page web et site web

  • site web

    est un ensemble de pages web regroupées entre elles de différentes façons qui partagent un nom de domaine.

  • Page web

    est Un document qui peut être affiché par un navigateur web tel que Mozilla Firefox…..elle est décrit Avec un langage de description tel que HTML5

CREATION D'UNE PAGE WEB HTML

    un document html est un fichier texte construit suivant une syntaxe précise, structuré par des balises, interprété dans un navigateur, créé dans un éditeur de pages web tel que vs code

  • les principes de langage html sont

    • les balises

      • /- html> ... /html>/

        Délimite le début et la fin du document

      • / head> ... /head> /

        Entête du document

      • /- body> ... /body>/

        Corps du document

    • Les titres et paragraphes

      • / -hn> ... hn> /

        Titre de niveau n :1 à 6

      • / p> ... p>

        paragraphe
